# LR-WGAN
The repository includes both the datasets and links to large preprocessed files required for reproducibility:

SKAB Dataset:
The file sensor.csv corresponds to the SKAB (Skoltech Anomaly Benchmark) dataset used in our experiments.
UCI Appliances Energy Prediction Dataset:
The file uci_energy_T_RH_weather_sensors.csv corresponds to the UCI Energy dataset, which includes indoor temperature, humidity, and weather-related sensor data.
LRMR Outputs (Low-Rank Matrix Recovery):
Due to size limitations, the LRMC (Low-Rank Matrix Completion) outputs are not fully stored in this repository:
The folder LRMR outputs contains a Google Drive link for the SKAB LRMC files.
The folder LRMC_ENERGY_WEATHER_SEQUENCES_* contains a Google Drive link for the UCI Energy LRMC files.

These external links are necessary to reproduce the imputation results, as LRMR outputs are used as inputs to the LR-WGAN model.
